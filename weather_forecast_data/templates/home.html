Weather Forecast Service

This project is an REST Service based web application that allows to retrieve information of the weather forecast for an specific city.

For a forecast summary use:

  • http://weather-information-api.herokuapp.com/weather/summary/[city]>/[date]/[hour_minutes]/
Example:
  • https://weather-information-api.herokuapp.com/weather/summary/berlin/20190526/1800/

For a forecast detailed use:

  • http://weather-information-api.herokuapp.com/weather/[temperature, pressure, humidity]/[city]>/[date]/[hour_minutes]
Example:
  • https://weather-information-api.herokuapp.com/weather/temperature/hamburg/20190526/1800/
